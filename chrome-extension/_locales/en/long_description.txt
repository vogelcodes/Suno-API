# Suno Tracks Exporter - Complete User Guide

## Overview
Suno Tracks Exporter is a powerful, feature-rich Chrome extension designed to help musicians, content creators, and music enthusiasts efficiently export and download all their music tracks created on Suno.com. Whether you're managing a large library of AI-generated music, backing up your creative work, or organizing tracks across multiple workspaces, this extension provides a seamless, automated solution for batch downloading your tracks in both MP3 and WAV formats with intelligent organization and progress tracking.

## Why Use Suno Tracks Exporter?

### The Problem
Suno.com is an incredible platform for creating AI-generated music, but manually downloading each track one by one is time-consuming and tedious. If you have dozens or hundreds of tracks across multiple workspaces, the manual process becomes impractical. Additionally, Suno's interface doesn't provide bulk download functionality, making it difficult to:
- Back up your entire music library
- Organize tracks by workspace
- Download tracks in your preferred format (MP3 or WAV)
- Manage large collections efficiently

### The Solution
Suno Tracks Exporter solves all these problems by providing:
- **Automated batch downloading** of all your tracks
- **Multi-workspace support** to fetch tracks from every workspace automatically
- **Format flexibility** with both MP3 and high-quality WAV options
- **Smart organization** with workspace grouping and custom filenames
- **Progress tracking** to monitor large download operations
- **Zero configuration** - works automatically once installed

## How It Works

### Installation Process

**Step 1: Install from Chrome Web Store**
1. Visit the Chrome Web Store
2. Search for "Suno Tracks Exporter"
3. Click "Add to Chrome"
4. Confirm installation
5. Extension appears in your toolbar

**Step 2: Initial Setup**
1. No additional configuration required!
2. Extension is ready to use immediately
3. First-time setup is automatic

**Step 3: First Use**
1. Visit suno.com and log in to your account
2. Navigate around the site (library, create page, etc.)
3. Extension automatically captures authentication token
4. Ready to use!

### Using the Extension

**Opening the Downloader**
1. Click the extension icon in your Chrome toolbar
2. Click "Open Downloader on Suno.com" button
3. Modal interface appears on Suno.com
4. Tracks begin loading automatically

**Token Management**
- Token is captured automatically as you browse
- If token is missing, click "Extract Token from Page"
- Extension navigates to library page to trigger API calls
- Token captured within seconds
- Status indicators show token state

### Downloading Tracks

**MP3 Download Process**
1. Select tracks you want to download:
   - Individual checkboxes for specific tracks
   - Workspace checkbox for all tracks in a workspace
   - Master checkbox for all tracks
2. Click "Download Selected MP3" button
3. Downloads begin immediately
4. Progress bar shows completion status
5. Files saved to default Chrome Downloads folder
6. Filenames follow convention: `workspace-trackname-trackid.mp3`

**WAV Download Process**
1. Select tracks you want to convert to WAV
2. Click "Convert & Download Selected WAV" button
3. Extension initiates conversion for each track:
   - Sends conversion request to Suno API
   - Polls for conversion completion
   - Monitors conversion status
4. Once converted, downloads WAV file automatically
5. Progress updates show conversion and download status
6. Files saved with `.wav` extension

**Individual Track Downloads**
- Click "MP3" button next to any track for instant download
- Click "WAV" button to convert and download that track
- Useful for single-track downloads
- Same filename convention applies

### Troubleshooting Guide

**Token Issues**
- **Problem**: "Token not found" error
- **Solution**: Click "Extract Token from Page" button
- **Alternative**: Navigate to different pages on Suno.com
- **Prevention**: Keep Suno.com tab open and active

**Missing Tracks**
- **Problem**: Some tracks not appearing
- **Solution**: Navigate to library page to trigger API calls
- **Check**: Ensure you're logged into correct account
- **Verify**: Check if tracks exist in Suno.com interface

**Download Errors**
- **Problem**: Downloads failing
- **Solution**: Check Chrome Downloads folder permissions
- **Verify**: Ensure sufficient disk space
- **Check**: Antivirus software not blocking downloads
- **Retry**: Use individual download buttons for failed tracks

**Conversion Failures**
- **Problem**: WAV conversion not completing
- **Solution**: Wait longer, conversions can take time
- **Retry**: Try converting individual tracks
- **Check**: Track may not support WAV conversion
- **Alternative**: Download MP3 version instead

**Performance Issues**
- **Problem**: Extension running slowly
- **Solution**: Close other browser tabs
- **Check**: System resources (CPU, memory)
- **Reduce**: Download smaller batches
- **Update**: Ensure latest extension version

**UI Issues**
- **Problem**: Modal not appearing
- **Solution**: Refresh Suno.com page
- **Check**: Extension enabled in Chrome
- **Verify**: On correct Suno.com page
- **Reload**: Reload extension in chrome://extensions/

## Technical Details

### Permissions Explained

**storage**
- Purpose: Saves authentication token and user preferences
- Scope: Local Chrome storage only
- Data: Token, device ID, preferences
- Privacy: All data stays on your device

**downloads**
- Purpose: Enables file downloads to your computer
- Scope: Chrome Downloads API
- Function: Manages download queue and filenames
- Security: Respects user download preferences

**tabs**
- Purpose: Allows interaction with Suno.com tabs
- Scope: Active Suno.com tabs only
- Function: Injects UI, sends messages
- Privacy: Only accesses Suno.com

**scripting**
- Purpose: Injects downloader interface into Suno.com
- Scope: suno.com domain only
- Function: Content script injection
- Security: Limited to specified domain

**webRequest**
- Purpose: Captures authentication tokens from API requests
- Scope: studio-api.prod.suno.com only
- Function: Intercepts authorization headers
- Privacy: Only captures tokens, no other data

### API Endpoints Used

**Workspace Listing**
- Endpoint: `GET /api/project/me`
- Purpose: Fetches all user workspaces
- Parameters: Pagination, sorting, filters
- Response: List of workspace objects

**Track Listing**
- Endpoint: `POST /api/feed/v3`
- Purpose: Fetches tracks with pagination
- Parameters: Workspace ID, cursor, limit
- Response: Tracks array with pagination info

**WAV Conversion Initiation**
- Endpoint: `POST /api/gen/{clip_id}/convert_wav/`
- Purpose: Starts WAV conversion process
- Parameters: Clip ID
- Response: 204 No Content (success)

**WAV File Polling**
- Endpoint: `GET /api/gen/{clip_id}/wav_file/`
- Purpose: Checks conversion status and gets WAV URL
- Parameters: Clip ID
- Response: WAV file URL or pending status
