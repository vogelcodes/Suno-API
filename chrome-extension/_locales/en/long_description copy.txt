# Suno Tracks Exporter - Complete Guide

## Overview
Suno Tracks Exporter is a powerful Chrome extension designed to help you efficiently export and download all your music tracks created on Suno.com. Whether you're a content creator, musician, or just someone who loves creating AI-generated music, this extension makes it easy to batch download your tracks in both MP3 and WAV formats, organized by workspace.

## How It Works

### Installation
1. Install the extension from the Chrome Web Store
2. The extension will appear in your Chrome toolbar
3. No additional setup required!

### First Use
1. Visit suno.com and log in to your account
2. Navigate around the site (library, create page, etc.) to allow token capture
3. Click the extension icon in your toolbar
4. Click "Open Downloader on Suno.com"
5. The extension will automatically fetch all your tracks from all workspaces

### Downloading Tracks

#### MP3 Downloads
1. Select the tracks you want to download (individual checkboxes or workspace-level selection)
2. Click "Download Selected MP3"
3. Files will be saved to your default Chrome Downloads folder
4. Progress is shown in real-time

#### WAV Downloads
1. Select the tracks you want to convert
2. Click "Convert & Download Selected WAV"
3. The extension will:
   - Initiate WAV conversion for each track
   - Poll for conversion completion
   - Download the WAV files automatically
4. This process takes longer but provides higher quality

### Workspace Management
- **View All Workspaces**: All your workspaces are listed automatically
- **Collapse/Expand**: Click workspace headers to collapse or expand
- **Select All**: Use the master checkbox to select all tracks
- **Workspace Selection**: Use workspace checkboxes to select all tracks in a workspace

### Filename Convention
Files are saved with the format: `workspace-trackname-trackid.extension`
- Example: `MyWorkspace-MyAwesomeSong-abc123def456.mp3`
- This ensures easy organization and prevents duplicates

## Tips & Best Practices

### For Best Performance
- **Automatic Downloads**: Enable automatic downloads in Chrome settings to avoid confirmation dialogs
  - Go to Chrome Settings ‚Üí Downloads
  - Turn OFF "Ask where to save each file before downloading"
- **Batch Size**: For large collections, download in smaller batches to avoid overwhelming your system
- **WAV Conversion**: WAV conversions take time - be patient, especially for large batches

### Organization Tips
- **Workspace Naming**: Use descriptive workspace names for easier file organization
- **Track Titles**: Keep track titles concise and descriptive
- **File Management**: Consider organizing downloaded files into folders by workspace

### Troubleshooting
- **Token Issues**: If downloads fail, click "Extract Token from Page" to refresh your authentication
- **Missing Tracks**: Navigate to different pages on Suno.com to trigger API calls
- **Download Errors**: Check your Chrome downloads folder permissions
- **Conversion Failures**: Some tracks may fail to convert - these will be marked in the progress

## Technical Details

### Permissions Explained
- **storage**: Saves your authentication token and preferences locally
- **downloads**: Enables file downloads to your computer
- **tabs**: Allows the extension to interact with Suno.com tabs
- **scripting**: Injects the downloader interface into Suno.com
- **webRequest**: Captures authentication tokens from API requests

### API Endpoints Used
- `/api/project/me`: Fetches all workspaces
- `/api/feed/v3`: Fetches tracks with pagination
- `/api/gen/{clip_id}/convert_wav/`: Initiates WAV conversion
- `/api/gen/{clip_id}/wav_file/`: Polls for WAV file URL
